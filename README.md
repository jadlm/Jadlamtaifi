# Jadlamtaifi

Portfolio project with a 3D floating HTML tag background animated using React Three Fiber and Drei.

## Notes

- The scene renders floating HTML-like code tags behind the page.
- Opacity has been softened so page content remains readable.

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/jadlm/Jadlamtaifi.git
```
